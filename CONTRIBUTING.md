# Contributing Guide

## 1. Introduction
Thank you for your interest in contributing to this project! All types of contributions are welcome.

## 2. Types of Contributions
We accept the following contributions:
- Bug reports (қате есептері)
- Bug fixes (қателерді түзету)
- Documentation improvements (құжаттаманы жақсарту)
- New features and enhancements (жетілдірулер)
- Ideas and suggestions (идеялар)

## 3. How to Contribute

### Step 1: Fork the repository
Click the "Fork" button on GitHub.

git clone https://github.com/your-username/repository-name.git
